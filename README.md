# Karakter-De-i-tirme
Verilen string ifade içerisindeki ilk ve son karakterin yerini değiştirip tekrar ekrana yazdıran console uygulamasını yazınız.  Örnek: Input: Merhaba Hello Algoritma x  Output: aerhabM oellH algoritmA x
